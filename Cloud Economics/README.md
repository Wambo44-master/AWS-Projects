# ☁ Cloud Economics


Welcome! This is my **Cloud Economics** knowledge center. It is made up of a collection of research, explanations and practical guides on the economies of AWS Cloud.

## Purpose

A key benefit of cloud computing is cost reduction achieved by pay-as-you-go pricing. This folder dives into  the **cost** of various AWS services and **how** to optimize them.
